# Conversor de Moedas - Desafio ONE

Este projeto foi desenvolvido como parte de um desafio do programa **Oracle Next Education (ONE)** em parceria com a **Alura**. O objetivo principal é colocar em prática conhecimentos de Java, consumo de APIs e manipulação de dados JSON.

---

## Descrição do Projeto
A aplicação é um conversor de moedas que utiliza a **ExchangeRate-API** para obter taxas de câmbio em tempo real. O sistema permite converter valores entre moedas especificadas previamente, garantindo dados atualizados e precisos diretamente de uma fonte externa.

### Tecnologias Utilizadas
* **Java** (JDK 17 ou superior)
* **HttpClient**: Biblioteca nativa do Java para requisições HTTP modernas.
* **Gson**: Biblioteca do Google utilizada para realizar o parse de objetos JSON.
* **ExchangeRate-API**: Interface de programação que fornece os dados de câmbio.

## Funcionalidades
* Conexão em tempo real com API externa.
* Extração dinâmica de taxas de conversão (acesso a chaves aninhadas no JSON).
* Conversão de valores com base na cotação atualizada.
